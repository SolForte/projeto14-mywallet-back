# My Wallet
My Wallet is a finance management application.

## About
This project is a web API. Implemented features are:
<ul>
  <li>Sign Up;</li>
  <li>Sign In;</li>
  <li>Get Transaction History;</li>
  <li>Add Expense/Revenue (Transaction).</li>
</ul>
